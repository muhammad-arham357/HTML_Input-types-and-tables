Terminologies related to the field of IT:

1. HTML
A markup language used to create the structure and content of web pages
2. AI (Artificial intelligence)
A type of computer science that helps machines learn, reason, and solve problems on their own, like a human
3. CSS
A coding language that allows you to control the layout, design, and visual appearance of websites, making them more attractive and user-friendly
4. Python
A high-level programming language used to create games, apps, websites, and software by writing simple and easy-to-read code
5. Java-script
A programming language used to add interactive to websites, making them more engaging and user-friendly.
6. Web scrapping
A technique used to automatically extract and collect data from websites, saving time and effort.
7. Machine learning
A way for computers to learn from data and improve their performance on tasks, without being explicitly programmed.
8. Web developing
The process of building and maintaining websites and web applications using programming languages, tools, and technologies.
9. Role of IT in future
Information Technology will play a vital role in shaping the future by enabling automation, innovation, and connectivity
10. Machine code
A set of instructions that a computer's processor can execute directly, written in binary code (0s and 1s) that the computer can understand.
